INSTALL INSTRUCTIONS:
Ensure that NPM and NodeJS are installed.
Execute "npm install"

LAUNCH INSTRUCTIONS:
After downloading the zip folder, extract it, and open a terminal on the folder. 
Ensure "npm install" has been executed
Then, execute node server.js
Once the server is running, open up a web browser to http://localhost:3000/curling.html


TESTING INSTRUCTIONS:
When the server is running, visit http://localhost:3000/curling.html
